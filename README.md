# Weather API Integration

## Description
This Python project fetches real-time weather information using the Open-Meteo API.

## Features
- Accepts city name as input
- Fetches current weather
- Displays temperature and wind speed
- Handles invalid city names

## Technologies Used
- Python
- Requests
- Open-Meteo API

## How to Run

1. Install the requests library:

pip install requests

2. Run the project:

python weather.py

3. Enter a city name.

## Sample Output

Enter city name: Bangalore

------ Weather Report ------
City: Bangalore
Temperature: 25.4 °C
Wind Speed: 8.7 km/h
